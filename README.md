# BarCrawlr

BarCrawlr is a mobile-first Amsterdam bar crawl planner. Answer a few questions about the group, mood, start time, pace, and preferred vibe, then get a timed route with stops, walking legs, booking cautions, and Google Maps links.

## Run locally

```bash
python3 -m http.server 5173
```

Open `http://localhost:5173`.

## Deploy as a website

This is a static site, so it can be served directly from GitHub Pages, Netlify, Vercel, or any static host.

For GitHub Pages, deploy from the repository root on the `main` branch.
